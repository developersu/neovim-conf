# What an awesome neovim config

Created for world's neovim configurations contest with intension to win the Internet!

Requirements:
* For vim-treesitter: `tree-sitter-cli`
* For JSON:`jq`
* For hex editing: `xxd`

nvim-treesitter installation process: `:TSInstall bash c css cpp go html java javascript json lua markdown markdown_inline python rust tsx typescript asm`

Plugins installation process: `:PlugInstall`
